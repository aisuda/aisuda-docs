老虎机抽奖

用老虎机滚动的方式，实现奖品的抽取。

属性​

数据​

奖品列表可以通过接口返回，也可以在编辑器里配置，支持新增、编辑、删除

接口返回格式如下：

{
    status: 0,
    data: [
        {
            id: '1',
            title: '选项1',
            picture: 'http://**'
        },
        {
            id: '2',
            title: '选项1',
            picture: 'http://**'
        }
    ]
}
