小程序登录

小程序授权登录按钮，可以实现静默登录，手机号获取。

属性​

数据​

登录接口主要是拿微信返回的code换取用户信息，具体参考微信官方文档

code模拟可用于浏览器端h5调试，需和后端约定登录成功失败。

登录接口入参：

{
    code: '**',
    appId: '**'
}

登录接口返回如下:

登录成功返回

{
    status: 0
}

登录失败返回

{
    status: -1
}

事件​

可以通过配置登录成功事件实现页面的刷新、跳转等。
