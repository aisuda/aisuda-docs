单选框

选项列表中选中的为默认选项，选项列表可通过fx绑定接口返回.

属性​

常规​

选项支持新增、编辑、删除，如需设置默认值直接选中即可，默认第一项是默认值。

编辑

选项列表支持通过fx动态绑定，例如通过自定义代码：

当然也可以通过api接口返回

接口格式为:

{
    status: 0,
    data: [
        {
            "label": "",
            "value": ""
        }
    ]
}

外观​

单选框样式​

支持选中颜色、选中字体色、显示样式(默认、纵向列表、标签展示)设置。

选项列表参数说明：

属性名属性标识类型说明
选项文本labelstring
选项值valuestring
默认选中defaultboolean多个选项只能配置一个default
