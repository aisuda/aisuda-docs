## 场景简介

通过`更新记录`节点可以实现更新实体中的记录，典型场景如下：

- 批量更新当前实体记录；

- 更新关联实体中的数据。

## 属性说明

| 属性                     | 说明                                                   |
| ------------------------ | ------------------------------------------------------ |
| 筛选条件                 | 设定更新记录的筛选条件，满足筛选条件的记录将会被更新。 |
| 新增 1:n、n:n 关联表记录 | 针对关联实体的更新，开启则表示在关联实体中新增记录。   |
| 更新字段                 | 设置需要更新的字段。                                   |
