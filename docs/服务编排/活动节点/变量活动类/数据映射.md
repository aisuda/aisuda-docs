## 场景简介

对已有变量中的数据进行映射，如对已有数据列表中的人员 ID 映射为人员姓名等，支持以下场景：

- 组织信息映射
- 静态值映射
- 对象数组映射

## 组织信息映射

将人员 ID 映射为组织信息、如邮箱、用户名称、部门等。

![](/img/服务编排/活动节点/变量活动类/数据映射/数据映射03.png)

## 静态值映射

将需要数据映射的变量中的静态值 A 映射为另一个静态值 B
