

## 场景简介

通过该节点，可以调用连接器服务。



## 属性概要

| 属性         | 说明                                                 |
| ------------ | ---------------------------------------------------- |
| 连接器       | 选择调用目标的连接器，支持从已经注册的连接器中选择。 |
| 方法         | 调用连接器的方法。                                   |
| 返回结果转换 | 对返回结果进行转换。                                 |
| 返回结果适配 | 通过代码进行返回结果的适配。                         |
