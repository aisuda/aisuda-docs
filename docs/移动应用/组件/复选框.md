选项列表中选中的为默认选项，选项列表可通过fx绑定接口返回。


![image.png](/img/移动应用/组件/checkbox-1.png)

使用方式与单选框相同 [单选框](./单选框)


选项列表参数说明：

| 属性名  | 属性标识                   | 类型     | 说明 |
| -----  | ------------------------- | ------- |   --- |
| 选项文本    | label                    | string  | 
| 选项值  | value                  | string  | 
| 默认选中 | default                | boolean | 多个选项可以配置多个default
