---
id: 纵向选项卡
---

用于实现常见的选项卡切换效果, 移动端纵向菜单导航等。

![image.png](/img/移动应用/组件/vertical-tab-section-1.png)


## 外观

 外观配置与横向选项卡唯一不同的是可以设置标签大小的展示行数，可以设置满屏高度。

![image.png](/img/移动应用/组件/vertical-tab-section-2.png)


## 选项列表字段

| 属性名  | 属性标识                   | 类型     | 
| -----  | ------------------------- | ------- |  
| 标题    | name                    | string  | 
| 点击事件 | action                   | object  |
